# Installation

##### Install Meteor
`curl https://install.meteor.com/ | sh` if Linux/Mac OSX or if Windows exectuable installer

##### Install Dependencies
`meteor npm install`

##### Run

`meteor`

Then navigate to `localhost:3000` in your browser of choice
